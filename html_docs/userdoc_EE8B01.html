User Commands

daily

Allows the user to collect a daily bonus can also activate with !daily

help

Posts a link to the help document for this bot can also activate with !help

members

Prints metrics about the server's player count can also activate with !members

nextrank

Displays how close the user is from reaching the next rank can also activate with !nextrank

richest

Displays a list of users ranked by money earned can also activate with !richest

whoami

Prints out user's discord name and id can also activate with !whoami

work

Allows the user to work to earn cookies can also activate with !work
